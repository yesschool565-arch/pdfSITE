# SmartTools Hub - All-in-One Utility Platform

A modern, fast, and fully frontend-based utility website featuring 25+ tools for PDF, Images, Text, and Security. Built using HTML, CSS, and Vanilla JavaScript.

## 🚀 Key Features
- **100% Client-Side**: All processing happens in your browser. No files are ever uploaded to a server.
- **Privacy-Focused**: Your data never leaves your computer.
- **Modern UI**: Clean, mobile-responsive design with dark/light mode toggle.
- **Fast & Lightweight**: Minimal dependencies, optimized for performance.

## 🛠️ Included Tools

### PDF Tools
- **Merge PDF**: Combine multiple PDFs into one.
- **Compress PDF**: Reduce file size (optimized in-browser).
- **JPG to PDF**: Convert images to professional PDF documents.
- **Split & Rotate**: coming soon.

### Image Tools
- **Resize & Compress**: Change dimensions and optimize file size.
- **Image Converter**: Convert between JPG, PNG, WEBP.
- **OCR Text**: Extract text from images using Tesseract.js.
- **Crop & Blur**: coming soon.

### Utility & Security
- **QR Code Generator**: Create custom QR codes for any link.
- **Password Generator**: Generate high-security random passwords.
- **Unit Converter**: Convert Length, Weight, Temp, etc.
- **Online Notepad**: Simple, autosaved notes using LocalStorage.

## 📦 Tech Stack
- **Languages**: HTML5, CSS3, JavaScript (ES6+)
- **Libraries**:
  - `pdf-lib` (PDF merging/modification)
  - `Tesseract.js` (Optical Character Recognition)
  - `Compressor.js` (Image optimization)
  - `QRCode.js` (QR generation)
  - `FontAwesome` (Icons)
  - `Google Fonts` (Typography)

1. Clone the repository.
2. Open `index.html` in any modern web browser.
